# Deployments API

Deployment routes are world-scoped.

## Endpoints

### List Deployments

```http
GET /api/worlds/:worldId/deployments?page=1&limit=20
```

### Create Deployment

```http
POST /api/worlds/:worldId/deployments
```

```json title="Request Body"
{
  "name": "Season 1",
  "description": "Runtime deployment",
  "mode": "upgradable",
  "targetChains": ["near-testnet"],
  "aiModelId": "openai/gpt-oss-120b",
  "onchain": {
    "stateChanges": ["wins", "speed_rating"],
    "result": ["winner"]
  },
  "isLLMPublic": false,
  "payToAddress": "6Q2...wallet",
  "firstBinding": {
    "eventId": 29,
    "eventVersion": 1
  },
  "sourceDeploymentId": 12
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Deployment name. |
| `description` | `string` | No | Description. |
| `mode` | `enum` | No | `upgradable | locked` (default `upgradable`). |
| `targetChains` | `enum[]` | No | `solana-devnet | solana-mainnet | near-testnet | near-mainnet`. |
| `aiModelId` | `string` | No | Selected model ID. |
| `onchain` | `object` | No | Field filter for on-chain push. |
| `onchain.stateChanges` | `string[]` | No | Allowed state-change keys. |
| `onchain.result` | `string[]` | No | Allowed result keys. |
| `isLLMPublic` | `boolean` | No | Whether LLM prompt/response data is public. |
| `payToAddress` | `string` | No | Payment receiver Solana address for paid public execution. |
| `firstBinding` | `object` | Yes | Initial bound event version. |
| `firstBinding.eventId` | `number` | Yes | Event ID. |
| `firstBinding.eventVersion` | `number` | Yes | Relative version number. |
| `sourceDeploymentId` | `number` | No | Clone entity state and bindings from source deployment. |

### Execute Event

```http
POST /api/worlds/:worldId/deployments/:deploymentId/execute
```

```json title="Request Body"
{
  "eventId": 29,
  "input": { "raceId": "RACE_001" }
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `eventId` | `number` | Yes | Bound event to execute. |
| `input` | `object` | No | Event input payload. |

### Lock Deployment

```http
POST /api/worlds/:worldId/deployments/:deploymentId/lock
```

### List Event History

```http
GET /api/worlds/:worldId/deployments/:deploymentId/history?page=1&limit=10
```

### List Event Bindings

```http
GET /api/worlds/:worldId/deployments/:deploymentId/event-bindings
```

### Create Event Binding

```http
POST /api/worlds/:worldId/deployments/:deploymentId/event-bindings
```

```json title="Request Body"
{
  "eventId": 29,
  "eventVersion": 2
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `eventId` | `number` | Yes | Event to bind. |
| `eventVersion` | `number` | Yes | Relative version number to bind. |

### List Entity Instances

```http
GET /api/worlds/:worldId/deployments/:deploymentId/entity-instances?page=1&limit=20
```

### Create Entity Instance

```http
POST /api/worlds/:worldId/deployments/:deploymentId/entity-instances
```

```json title="Request Body"
{
  "entitySchemaId": 53,
  "instanceId": "HORSE_1",
  "state": {
    "name": "Midnight Comet",
    "speed_rating": 0.85
  }
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `entitySchemaId` | `number` | Yes | Entity schema for the instance. |
| `instanceId` | `string` | Yes | Unique per deployment+schema. |
| `state` | `object` | No | Initial state. Defaults to `{}`. |

## Public Execution

Public execution is available via app route:

```http
POST /app/deployments/:address/execute
```

```json title="Request Body"
{
  "eventId": 29,
  "input": { "raceId": "RACE_001" },
  "wallet": "<solana-wallet>",
  "signature": "<base58-signature>",
  "message": "narrativeprotocol:<deploymentAddress>:<eventId>:<timestamp>"
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `eventId` | `number` | Yes | Bound event to execute. |
| `input` | `object` | No | Event input payload. |
| `wallet` | `string` | Yes | Signer wallet (base58 public key). |
| `signature` | `string` | Yes | Signature over `message`. |
| `message` | `string` | Yes | `narrativeprotocol:{deploymentAddress}:{eventId}:{timestamp}`. |

Rules:

- Event must allow public execution.
- If `executionSettings.priceUsd > 0`, provide valid `X-Payment` header (x402).
