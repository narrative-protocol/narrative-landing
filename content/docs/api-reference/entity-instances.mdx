# Entity Instances API

Entity instance routes are deployment-scoped under worlds.

## Endpoints

### List Entity Instances

```http
GET /api/worlds/:worldId/deployments/:deploymentId/entity-instances
```

### Create Entity Instance

```http
POST /api/worlds/:worldId/deployments/:deploymentId/entity-instances
```

```json title="Request Body"
{
  "entitySchemaId": 53,
  "instanceId": "HORSE_1",
  "state": {
    "name": "Midnight Comet",
    "speed_rating": 0.85
  }
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `entitySchemaId` | `number` | Yes | Entity schema ID. |
| `instanceId` | `string` | Yes | Instance key unique in deployment+schema. |
| `state` | `object` | No | Initial state. Defaults to `{}`. |

## Example

```bash
curl -X POST https://api.narrativeprotocol.com/api/worlds/39/deployments/28/entity-instances \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "entitySchemaId": 53,
    "instanceId": "HORSE_1",
    "state": {
      "name": "Midnight Comet",
      "speed_rating": 0.85
    }
  }'
```
