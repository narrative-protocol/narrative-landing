# Entity Schemas API

Entity schema routes are world-scoped.

## Endpoints

### List Entity Schemas

```http
GET /api/worlds/:worldId/entity-schemas
```

### Create Entity Schema

```http
POST /api/worlds/:worldId/entity-schemas
```

### List Attributes for a Schema

```http
GET /api/worlds/:worldId/entity-schemas/:schemaId/attributes
```

### Create Attribute

```http
POST /api/worlds/:worldId/entity-schemas/:schemaId/attributes
```

### Get Attribute by Name

```http
GET /api/worlds/:worldId/entity-schemas/:schemaId/attributes/:name
```

### Update Attribute by Name

```http
PUT /api/worlds/:worldId/entity-schemas/:schemaId/attributes/:name
```

### Delete Attribute by Name

```http
DELETE /api/worlds/:worldId/entity-schemas/:schemaId/attributes/:name
```

## Examples

### Create Entity Schema

```bash
curl -X POST https://api.narrativeprotocol.com/api/worlds/39/entity-schemas \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "name": "horse",
    "description": "A racing horse"
  }'
```

### Add Attribute

```bash
curl -X POST https://api.narrativeprotocol.com/api/worlds/39/entity-schemas/53/attributes \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "name": "speed_rating",
    "type": "float",
    "constraints": { "min": 0, "max": 1 },
    "defaultValue": 0.5
  }'
```
