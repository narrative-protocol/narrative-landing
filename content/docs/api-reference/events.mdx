# Events API

Event routes are world-scoped.

## Endpoints

### List Events

```http
GET /api/worlds/:worldId/events
```

### Create Event

```http
POST /api/worlds/:worldId/events
```

### List Event Versions

```http
GET /api/worlds/:worldId/events/:eventId/event-versions
```

### Create Event Version

```http
POST /api/worlds/:worldId/events/:eventId/event-versions
```

### Publish Event Version

```http
POST /api/worlds/:worldId/events/:eventId/event-versions/:version/publish
```

## Event Version Fields

- `inputSchema`
- `outputSchema`
- `readEntities` (`[{ schemaId, filter? }]`)
- `entityChangeSettings` (`{ [schemaName]: "partial" | "full" | "append" }`)
- `behaviorPrompt`
- `mutationSettings` (`{ mode: "ai" | "direct", template? }`)
- `executionSettings` (`{ visibility: "admin" | "public" | "cron", priceUsd?, cron? }`)

## Example

```bash
curl -X POST https://api.narrativeprotocol.com/api/worlds/39/events \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "name": "race_result",
    "description": "Resolves a race and updates horse stats",
    "firstVersion": {
      "inputSchema": { "raceId": "string" },
      "readEntities": [{ "schemaId": 53 }],
      "entityChangeSettings": { "horse": "partial" },
      "outputSchema": { "winner": "string", "time": "string" },
      "behaviorPrompt": "Determine the race winner based on horse speed ratings.",
      "mutationSettings": { "mode": "ai" },
      "executionSettings": { "visibility": "admin" }
    }
  }'
```
