# Worlds API

Create and manage worlds in Narrative Protocol.

## Endpoints

### List Worlds

```http
GET /api/worlds
```

### Create World

```http
POST /api/worlds
```

### Get World

```http
GET /api/worlds/:id
```

### Update World

```http
PUT /api/worlds/:id
```

### Delete World

```http
DELETE /api/worlds/:id
```

## Examples

### Create World

```bash
curl -X POST https://api.narrativeprotocol.com/api/worlds \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "name": "Horse Racing",
    "description": "A horse racing simulation world",
    "domainTags": ["sports", "simulation"],
    "promptSeed": "This world simulates realistic horse racing events."
  }'
```
