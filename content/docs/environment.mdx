# Endpoint & Environments

This guide covers the available endpoint and environments for developing with Narrative Protocol.

## Studio

Use the following endpoint to access the studio to register, add credits, and create API keys.

```bash title="Studio"
https://studio.narrativeprotocol.com
```

## API Endpoint

Use the following API endpoint for all requests:

```bash title="API Endpoint"
https://api.narrativeprotocol.com
```
